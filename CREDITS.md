# Credits — LightPropagation

Interactive simulation of how light propagates through polarizers and wave plates — Intro, Polarization, Wave Plates, and Lab screens.

The simulation is a faithful port of **[EMANIM](https://emanim.szialab.org)** (Electromagnetic
Waves Animated) by **András Szilágyi** — web app (<https://emanim.szialab.org>) and classic Python
